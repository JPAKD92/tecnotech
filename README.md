# TECNOTECH Presupuestos

App web estática para generar presupuestos en PDF desde el celular.

## Archivos
- `index.html`
- `styles.css`
- `app.js`
- `assets/logo.png`

## Cómo usar
1. Subí todo el contenido a un repositorio de GitHub.
2. Activá GitHub Pages.
3. Abrí la URL publicada desde el celular.
4. Completá los datos y tocá **GENERAR PDF**.

## Observaciones
- No usa backend.
- No guarda historial.
- El PDF se descarga en el dispositivo.
- El logo se toma desde `assets/logo.png`.
